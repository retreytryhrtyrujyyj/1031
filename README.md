# 宠物商城（后台）

## 项目简介
宠物商城前台系统配套的后台系统，包括宠物管理、订单管理、评价管理、用户管理等。对9张数据库表进行增删改查操作，适合毕业设计、课程设计、大作业等。

## 技术介绍
Java/SSM框架/JSP/Mysql数据库

## 开发工具
Eclipse或者MyEclipse或者IDEA Navicat

## 项目资料包含
1. 前后端源代码
2. 开题报告（任务书）+ 配套论文（一万两千字以上）+ 答辩PPT
3. 远程控屏调试（向日葵或者toDesk控屏）
4. 一键启动项目（双击运行演示，无需搭建复杂环境）
5. 项目讲解视频教程
